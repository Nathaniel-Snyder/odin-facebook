== odin-facebook

==== The Odin Project's Ruby on Rails Course

*Lesson:* Final Project, can be found at http://www.theodinproject.com/ruby-on-rails/final-project

*Objective:* Make a facebook-clone with the following functionality:
*   New user registration
*   Sign-in with Facebook
*   Send and accept friend requests
*   Publish posts
*   Like and comment on posts
---

Ruby version: 2.2.1

Postgresql database

Ran on Heroku at https://agile-island-8043.herokuapp.com/
